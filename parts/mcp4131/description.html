Libopenstm32l0x2

STM32L0x2 peripheral library

Microchip MCP4131

The MCP4131 is a 7-bit digital potentiometer. It has an SPI interface.

SPI interface

AddressFunction
0x00Volatile Wiper 0
0x01Volatile Wiper 1
0x04Volatile TCON Register
0x05Status Register
C1:C0CommandType
00Write Data16-bit
01Increment8-bit
10Decrement8-bit
11Read Data16-bit
