Libopenstm32l0x2

STM32L0x2 peripheral library

Microchip 25LC640A

The 25LC640A is a 64 Kbit (8192 x 8-bit) EEPROM. It has an SPI interface.

SPI interface

Instruction

NameFormatDescription
READ0000 0011Read data from memory array beginning at selected address
WRITE0000 0010Write data to memory array beginning at selected address
WRDI0000 0100Reset the write enable latch
WREN0000 0110Set the write enable latch
RDSR0000 0101Read STATUS register
WRSR0000 0001Write STATUS register
